SDK ENVIRONMENT VARIABLES (GovernanceDecorator)
================================================================================

Required (at least one):
  GOV_ORG_ID        Organization ID
  GOV_ORG_NAME      Organization name (fallback if GOV_ORG_ID not set)

Optional:
  GOV_PROJECT_ID      Project ID
  GOV_PROJECT_NAME    Project name
  GOV_TOOL_NAME       Tool label shown in dashboards         default: "unknown"
  GOV_ENDPOINT        Platform base URL                      default: "http://localhost:8000"
  GOV_API_KEY         Bearer token for authentication
  GOV_TOOL_VERSION    Semver for this tool                   default: "1.0.0"
  GOV_ENV             Execution environment label            default: "production"
  GOV_DETECT_PII      Enable PII detection                   default: "true"
  GOV_REDACT_PII      Replace PII with [REDACTED] in previews default: "true"
  GOV_CAPTURE_IO      Store input/output previews            default: "true"
  GOV_MAX_PREVIEW     Max chars per preview string           default: "500"
  GOV_BATCH_SIZE      Events to buffer before sending        default: "1"
